# Заключение

Мы подошли к завершению курса «JS: React». Вы познакомились с синтаксисом JSX, научились создавать компоненты, познакомились с жизненным циклом компонента. Этих знаний достаточно, чтобы создавать простые приложения.

Для создания более сложных приложений нужно научиться правильно организовывать состояние. Экосистема реакта содержит различные библиотеки, решающие эту задачу. Их изучение идет в следующих курсах по реакту.

Функциональные компоненты практически вытеснили классовые компоненты. Курс JS: React Hooks познакомит ближе с функциональными компонентами.

В курсе React: Redux Toolkit изучаются подходы в организации состояния и работы с ним, курс раскрывает подходы для создания сложных приложений на реакте и подойдет для тех, кто хочет ближе познакомиться с построением архитектуры react-приложений.
