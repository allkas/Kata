# Dynamic-Form

## Задание

#  JavaScript: Dynamic-Form 

JS: React

Начать испытание 

В этом задании вам предстоит разработать React компонент динамической формы, которая позволяет добавлять, удалять и изменять поля ввода. Компонент должен отображать введенные значения полей в реальном времени.

## Требования:

  * Добавление полей: пользователь может добавлять новые поля ввода с помощью кнопки "Add Field".
  * Удаление полей: каждое поле ввода имеет кнопку "Remove", которая позволяет удалить соответствующее поле.
  * Изменение значений: пользователь может вводить значения в поля ввода, и они должны отображаться в реальном времени.
  * Отображение введенных значений: введенные значения каждого поля отображаются ниже формы сразу же после ввода.
  * Уникальные индексы полей: каждое добавленное поле должно иметь уникальный индекс, который формируется автоматически. Каждый новый id вычисляется как последний id + 1. Если fields пустой, то новый id будет 1. После удаления поля оставшиеся элементы должны сохранить свои id без изменений. Например, если у нас были поля с id 1, 2, 3 и удалили поле с id 2, то оставшиеся поля должны иметь id 1 и 3.
