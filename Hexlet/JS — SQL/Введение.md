# Введение

В этом курсе мы будем изучать работу с базами данных в Node.js. На примере работы с PostgreSQL, вы научитесь создавать запросы, обновлять данные, работать с транзакциями и многое другое. После завершения этого курса вы сможете эффективно использовать SQL в ваших Node.js проектах.

В этом курсе вы научитесь работать с базами данных:

  * Подключаться к базе данных
  * Подготавливать и выполнять запросы
  * Использовать плейсхолдеры для защиты от SQL-инъекций
  * Работать с транзакциями
