# Безопасная регистрация

## Задание

#  JavaScript: Безопасная регистрация 

JS: Погружаясь в классы

Начать испытание 

В этом задании вам предстоит разработать функцию регистрации пользователя с надежной обработкой возможных ошибок. Для этого нужно будет создать систему пользовательских исключений.

### src/RegistrationError.js

Опишите базовый класс исключений `RegistrationError`, который будет наследовать от стандартного класса: `Error`. В конструкторе вызовите конструктор родительского класса, передав ему сообщение об ошибке.

### src/InvalidEmailError.js, PasswordsNotMatchError.js, UserExistsError.js

Опишите подклассы исключений. Каждый подкласс должен вызывать конструктор родительского класса с информативным сообщением об ошибке и устанавливать свойство `name` равным названию класса.

### src/registerUser.js

Реализуйте функцию `registerUser`, которая будет имитировать процесс регистрации пользователя. Функция принимает на вход 4 параметра: `users`, `email`, `password`, `confirmPassword` и выполняет соответствующие проверки. Если все в порядке, то функция возвращает обновленный массив пользователей, в противном случае выбрасывает соответствующее сообщение об ошибке: 

  * валидация `email`. Если `еmail` невалидный, выбрасывается ошибка `InvalidEmailError`
  * проверка, что пароли совпадают. Если пароли не совпадают, то выбрасывается исключение `PasswordsNotMatchError`
  * проверка на существование пользователя с таким `email`. Если пользователь существует, то выбрасывается исключение `UserExistsError`


    
    
    const users = ['test@example.com', test2@example.com];
    
    const email = "test@example.com";
    const password = "password123";
    const confirmPassword = "password123";
    
    try {
      registerUser(users, email, password, confirmPassword);
    } catch (error) {
      if (error instanceof RegistrationError) {
        console.error(`Ошибка регистрации: ${error.message}`); //Пользователь с таким email уже существует
      } else {
        console.error(`Произошла ошибка: ${error}`);
      }
