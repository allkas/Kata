# Гибкий логгер

## Задание

#  JavaScript: Гибкий логгер 

JS: Погружаясь в классы 1 сообщение

Начать испытание 

В этой практике мы реализуем систему логирования, которая может записывать сообщения различной степени важности в разные места назначения. 

## src/Logger.js

Создайте базовый класс `Logger`. Он принимает опциональный аргумент, определяющий уровень важности сообщения, которое будут логироваться (значение по умолчанию - `'INFO'`). 

Инициализируйте свойство `levels`, которое ставит в соответствие каждому уровню логирования числовое значение, что позволит удобно сравнивать уровни важности. Определите 4 уровня важности - `debug`, `info`, `warning`, `error` со значениями от 1 до 4 соответственно. Чем выше значение, тем выше уровень важности сообщения. Если переданный уровень важнее или равен, то вызывается метод `_writeMessage`.

Реализуйте шаблонный метод `log`, который будет являться основным методом для записи сообщений. Метод принимает на вход 2 параметра - сообщение и уровень важности.

Объявите абстрактный метод `_writeMessage`, его нужно будет переопределить в подклассах и описать конкретный способ записи сообщения (в консоль или в файл).

Опишите методы `debug`, `info`, `warning`, `error`, которые по сути являются обертками над методом `log` и упрощают запись сообщений с определенным уровнем важности.

## src/ConsoleLogger.js

Реализуйте логирование сообщений в консоль.
    
    
    const consoleLogger = new ConsoleLogger();
    
    const logger = new ConsoleLogger();
    logger.log('some message...'); // [INFO]: some message...
    logger.debug('debugging...'); // [INFO]: some information...
    logger.info('some information...'); // сообщение не будет выведено, т.к. уровень низкий
    logger.warning('some warning...'); // [WARNING]: some warning...
    

## src/FileLogger.js

Реализуйте логирование сообщений в файл. Каждое новое сообщение сохраняется на новой строке.
    
    
    const fileLogger = new FileLogger('WARNING', 'app.log'); 
    fileLogger.info('some information...'); // не будет записано в файл, т.к. уровень низкий
    fileLogger.warning('some warning...');
    fileLogger.error('error...'); 
    
    // в результате содержимое файла app.log будет следующим:
    // 2024-06-06T17:17:47.116Z [WARNING]: some warning...
    // 2024-06-06T17:17:47.117Z [ERROR]: error...
