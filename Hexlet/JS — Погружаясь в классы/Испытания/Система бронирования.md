# Система бронирования

## Задание

#  JavaScript: Система бронирования 

JS: Погружаясь в классы 1 сообщение

Начать испытание 

В этом задании будем разрабатывать систему бронирования билетов, которая может обрабатывать различные типы билетов: авиабилеты, ж/д билеты и кинобилеты.

### src/Ticket.js

Опишите базовый класс для всех типов билетов. Класс содержит следующие свойства для всех типов билетов: `number`, `price`, `date`. Реализуйте метод `getInfo()`, который возвращает строку с информацией о билете в следующем формате: "Номер билет: 123, Цена: 12500, Дата: 2024-03-13"

### src/AirplaneTicket.js, TrainTicket.js, CinemaTicket.js

Опишите подклассы для каждого типа билета. Каждый подкласс должен вызывать конструктор родительского класса для общих свойств, а также реализовывать собственные свойства, характерные для данного типа билета:

  * `AirplaneTicket`: название авиакомпании (`airline`), номер места (`seatNumber`), класс обслуживания (`serviceClass`)
  * `TrainTicket`: номер поезда (`trainNumber`), номер места (`seatNumber`), тип вагона (`wagonType`)
  * `CinemaTicket`: название фильма (`movieTitle`), номер места (`seatNumber`), номер зала (`hallNumber`)



Переопределите метод `getInfo()` родительского класса для вывода полной информации о билете, включая общие свойства и свойства, специфичные для данного типа.

### src/BookingSystem.js

Опишите класс `BookingSystem` для управления бронированием билетов. Методы класса должны корректно работать с любыми типами билетов, не зная их конкретный тип. Реализуйте следующие методы:

  * `bookTicket(ticket)` \- принимает объект типа `Ticket` (или любого его подкласса) и добавляет его в систему
  * `cancelBooking(ticketNumber)` \- принимает номер билета и отменяет бронирование, удаляя билет из системы. Возвращает `true` при успешной отмене, либо `false`, если билет с таким номеров в системе не найден
  * `getTicketInfo(ticketNumber)` \- принимает номер билета, находит его в системе и возвращает информацию о нем, либо возвращает `null`, если билет не найден


    
    
    const bookingSystem = new BookingSystem();
    
    const airplaneTicket = new AirplaneTicket(
      1,
      15000,
      "2024-03-10",
      "Аэрофлот",
      "12A",
      "Бизнес"
    );
    const trainTicket = new TrainTicket(
      2,
      5000,
      "2024-03-15",
      "010A",
      "35",
      "Купе"
    );
    const cinemaTicket = new CinemaTicket(
      3,
      1000,
      "2024-03-20",
      "Фильм",
      "7",
      "1"
    );
    
    bookingSystem.bookTicket(airplaneTicket);
    bookingSystem.bookTicket(trainTicket);
    bookingSystem.bookTicket(cinemaTicket);
    
    bookingSystem.getTicketInfo(2); // => 
    // TrainTicket {
    //   number: 2,
    //   price: 5000,
    //   date: '2024-03-15',
    //   trainNumber: '010A',
    //   seatNumber: '35',
    //   wagonType: 'Купе'
    // }
    
    bookingSystem.getTicketInfo(2).getInfo()); // => 
    // Номер билета: 2, Цена: 5000, Дата: 2024-03-15, Поезд: 010A, Место: 35, Вагон: Купе
    
    bookingSystem.cancelBooking(2); // true
    bookingSystem.getTicketInfo(2); // null
