## scr/main/java/io/hexlet/Random.java

Реализуйте генератор случайных чисел, представленный классом `Random`. Конструктор класса принимает на вход seed – начальное число генератора псевдослучайных чисел. Интерфейс генератора включает в себя две функции:

  * `getNext()` — метод, возвращающий новое случайное число
  * `reset()` — метод, сбрасывающий генератор на начальное значение


    
    
    var seq = new Random(100);
    var result1 = seq.getNext();
    var result2 = seq.getNext();
    
    result1 != result2; // true
    
    seq.reset();
    
    var result21 = seq.getNext();
    var result22 = seq.getNext();
    
    result1 == result21; // true
    result2 == result22; // true
    

### Подсказки

Простейший способ реализовать случайные числа — линейный конгруэнтный метод.
