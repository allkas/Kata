## scr/main/java/io/hexlet/Square.java

Реализуйте класс `Square` для описания квадратов. У квадрата есть только одно свойство — сторона, представленная числом типа `double`. Реализуйте в классе метод `getSide()`, возвращающий значение стороны квадрата.
    
    
    var square = new Square(16);
    square.getSide(); // 16
    

## scr/main/java/io/hexlet/SquareGenerator.java

Реализуйте класс `SquareGenerator` с публичным статическим методом `generate()`. Метод принимает два параметра – сторону квадрата и количество экземпляров, которые нужно создать. Метод должен вернуть массив из квадратов
    
    
    // Создаем массив из трех квадратов со стороной 5
    Square[] squares = SquareGenerator.generate(5, 3);
    
    // [new Square(5), new Square(5), new Square(5)]
