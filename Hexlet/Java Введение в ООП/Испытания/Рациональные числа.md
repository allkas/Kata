В этом испытании мы рассмотрим рациональные числа и базовые операции над ними. Вам предстоит реализовать класс для работы с рациональными числами. Рациональным называют число, которое может быть представлено в виде дроби a/b, где a — это числитель дроби, b — знаменатель дроби. Для рациональных чисел базовыми операциями будем считать сложение и вычитание.

## scr/main/java/io/hexlet/Rational.java

Реализуйте класс `Rational`, который представляет собой рациональное число. Конструктор класса принимает два аргумента – числитель и знаменатель числа. Класс должен содержать следующие интерфейсные методы:

  * Селектор `getNumer()` \- возвращает числитель
  * Селектор `getDenom()` \- возвращает знаменатель
  * Метод `add()` \- складывает рациональное число с другим, переданным в метод
  * Метод `sub()` \- вычитает из рационального числа другое число, переданное в метод
  * Метод `toString()` – возвращает строковое представление рационального числа в формате числитель/знаменатель



Методы `add()` и `sub()` должны возвращать новое рациональное число, а не изменять текущее. Не забудьте реализовать нормализацию дробей удобным для вас способом.
    
    
    var rat1 = new Rational(3, 9);
    
    rat1.getNumer(); // 1
    rat1.getDenom(); // 3
    rat1.toString(); // "1/3"
    
    var rat2 = new Rational(10, 3);
    var sum = rat1.add(rat2);
    sum.toString(); // "11/3"
    
    var sub = rat1.sub(rat2);
    sub.toString(); // "-3/1"
    

### Подсказки

Для нормализации дроби вам потребуется разделить числитель и знаменатель дроби на их наибольший общий делитель (НОД). Для поиска НОД двух чисел используйте статический метод `gcd()`
    
    
    gcd(3, 9); // 3
    gcd(10, 25); // 5
    

Этот метод уже определен и импортирован в упражнении
