## src/main/java/io/hexlet/model/BankAccount.java

В файле создайте класс `BankAccount`, который будет представлять собой банковский счет. У счета есть три свойства:

  * `accountNumber` — номер счета, число типа `long`
  * `accountCurrency` — валюта, в которой открыт счет, строка
  * `balance` — количество денег на счете в валюте счета, число типа `double`



Добавьте в класс конструктор со всеми полями и соответствующие геттеры

Для работы со счетом предусмотрены два метода:

  * Метод `deposit()` зачисляет деньги на счет. Принимает два параметра — валюту денег и их количество
  * Метод `withdraw()` списывает деньги со счета. Также принимает два параметра — валюту денег и их количество



Так как баланс счета содержит количество денег в валюте счета, то если валюта счета и денег не совпадает, перед зачислением и снятием их нужно конвертировать в валюту счета. Для простоты условимся, что наш аккаунт поддерживает только две валюты — _eur_ и _usd_. Ставка для обмена: 1 eur = 1.2 usd
    
    
    var account = new BankAccount(123, "eur", 10);
    
    account.deposit("eur", 10);
    account.getBalance(); // 20
    
    account.deposit("usd", 12);
    account.getBalance(); // 30
    
    account.withdraw("usd", 24);
    account.getBalance(); // 10
    
    account.withdraw("eur", 10);
    account.getBalance(); // 0
    

Не забудьте учесть инварианты. Баланс счета при снятии денег не должен уходить в минус. Если при снятии на счете недостаточно средств, должно быть выброшено исключение `IllegalArgumentException` с сообщением _Insufficient funds_. Баланс счета при этом должен остаться прежним
    
    
    var account = new BankAccount(123, "eur", 10);
    account.withdraw("eur", 20); // Ошибка!
    account.getBalance(); // 10
