## src/main/java/io/hexlet/model/Triangle.java

В файле создайте класс `Triangle`, который будет представлять собой треугольник. Конструктор класса принимает в качестве параметров три стороны треугольника — числа типа `double`. Добавьте в класс геттеры для всех сторон
    
    
    var figure = new Triangle(2, 3, 4);
    figure.getSideA(); // 2
    figure.getSideB(); // 3
    figure.getSideC(); // 4
    

Не забудьте учесть инварианты. Треугольник может существовать только в том случае, если сумма длин любых двух его сторон больше третьей. Если условие не выполняется, такой треугольник не существует. В этом случае должно быть выброшено исключение `IllegalArgumentException`
    
    
    var figure = new Triangle(1, 2, 3); // Ошибка!
