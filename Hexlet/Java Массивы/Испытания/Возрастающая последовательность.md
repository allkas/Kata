## src/main/java/io/hexlet/App.java

Реализуйте публичный статический метод `isContinuousSequence()`, который проверяет, является ли последовательность целых чисел в переданном массиве возрастающей непрерывно (не имеющей пропусков чисел). Например, последовательность {4, 5, 6, 7} — непрерывная, а {0, 1, 3} — нет. Последовательность может начинаться с любого числа, главное условие — отсутствие пропусков чисел. Последовательность из одного числа не может считаться возрастающей.
    
    
    int[] numbers1 = {10, 11, 12, 13};
    App.isContinuousSequence(numbers1); // true
    
    int[] numbers2 = {-5, -4, -3};
    App.isContinuousSequence(numbers2); // true
    
    int[] numbers3 = {10, 11, 12, 14};
    App.isContinuousSequence(numbers3); // false
    
    int[] numbers4 = {1, 2, 2, 3};
    App.isContinuousSequence(numbers4); // false
    
    int[] numbers5 = {5};
    App.isContinuousSequence(numbers5); // false
    
    int[] numbers6 = {};
    App.isContinuousSequence(numbers6); // false
