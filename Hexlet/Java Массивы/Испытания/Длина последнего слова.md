Реализуйте публичный статический метод `getLastWordLength()`, который возвращает длину последнего слова переданной на вход строки. Словом считается любая последовательность, не содержащая пробелов.
    
    
    App.getLastWordLength(""); // 0
    
    App.getLastWordLength("man in BlacK"); // 5
    
    App.getLastWordLength("hello, world!  "); // 6
