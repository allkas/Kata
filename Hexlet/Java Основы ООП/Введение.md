# Введение

Java – объектно-ориентированный язык программирования. Код в Java представляет собой набор классов и объектов, которые взаимодействуют друг с другом.

Объектно-ориентированное программирование – большая тема, которая пронизывает всю разработку на Java. В этом курсе мы продолжим знакомство с ООП и углубим свои знания в этом направлении. Мы рассмотрим такие основные темы:

  * Классы и объекты
  * Интерфейсы
  * Полиморфизм
  * Наследование
  * Обработка ошибок



Эти темы крайне важны даже для начинающих разработчиков на Java, потому что с этими понятиями они начинают сталкиваться с первых дней на новой работе. С другой стороны, требуется немало времени перед тем, как вы сможете действительно качественно использовать изучаемые подходы и техники.
