# Введение

В этом курсе мы рассмотрим продвинутые темы языка Java и важнейшие аспекты программирования в целом. Мы затронем следующие темы:

  * Многопоточность
  * Асинхронность
  * Контейнеризация



## Как подготовиться к этому курсу

Курс ориентирован на тех, кто уже хорошо знаком с языком программирования Java и хотел бы углубить свои знания. Курс опирается на темы, рассмотренные в предыдущих курсах:

  * Java: Основы ООП
  * Java: Корпоративные приложения на Spring Boot
