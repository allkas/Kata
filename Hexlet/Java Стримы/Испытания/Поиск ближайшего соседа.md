## scr/main/java/io/hexlet/App.java

В классе `App` реализуйте публичный статический метод `findIndexOfNearest()`, который принимает на вход список `List` целых чисел и искомое число. Задача метода — найти в списке ближайшее число к искомому и вернуть его индекс в списке.

Если в списке содержится несколько чисел, одновременно являющихся ближайшими к искомому числу, то возвращается наименьший из индексов ближайших чисел.
    
    
    App.findIndexOfNearest(new ArrayList<Integer>(), 0); // -1
    
    List<Integer> coll1 = List.of(15, 10, 3, 4);
    App.findIndexOfNearest(coll1, 0); // 2
    
    List<Integer> coll2 = List.of(7, 5, 3, 2);
    App.findIndexOfNearest(coll2, 4); // 1
    
    List<Integer> coll3 = List.of(7, 5, 4, 4, 3);
    App.findIndexOfNearest(coll3, 4); // 2
