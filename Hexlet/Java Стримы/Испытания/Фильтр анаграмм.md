Анаграммы — это слова, которые состоят из одинаковых букв. Например:

  * спаниель — апельсин
  * карат — карта — катар
  * топор — ропот — отпор



## scr/main/java/io/hexlet/App.java

В классе `App` реализуйте публичный статический метод `filterAnagram()`, который находит все анаграммы слова. Метод принимает исходное слово и список для проверки (список `List` строк). Метод должен вернуть список `List` всех анаграмм. Если в списке слов отсутствуют анаграммы, то возвращается пустой список.
    
    
    List<String> coll1 = List.of("aabb", "abcd", "bbaa", "dada");
    List<String> result1 = App.filterAnagram("abba", coll1);
    System.out.println(result1); // ["aabb", "bbaa"]
    
    List<String> coll2 = List.of("crazer", "carer", "racar", "caers", "racer");
    List<String> result2 = App.filterAnagram("racer", coll2);
    System.out.println(result2); // ["carer", "racer"]
    
    List<String> coll3 = List.of("lazing", "lazy", "lacer");
    List<String> result3 = App.filterAnagram("laser", coll3);
    System.out.println(result3); // []
