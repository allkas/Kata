# Футбольная лига

## Задание

SQL: Join

Начать испытание 

При проведении Чемпионата Европы о футболу собирается квалификационная группа из нескольких команд. Каждая из этих команд играет с каждой два матча - один домашний, другой гостевой.

В этом задании вам нужно из таблицы _teams_ собрать список матчей, которые будут проведены в квалификационной группе.

Таблица _teams_ содержит информацию о командах
    
    
    | id         | name          |
    |------------|---------------|
    | 1          | Барселона     |
    | 2          | ...           |
    

## solution.sql

Создайте запрос, который соберет будущие матчи команды. Команды играют два раза - на своем поле и в гостях. Команды отсортированы по первой и второй колонке по возрастанию.

Для команд _Милан_ , _Спартак_ , _Зенит_ результат будет следующий:
    
    
    | home_team | away_team |
    | --------- | --------- |
    | Зенит     | Милан     |
    | Зенит     | Спартак   |
    | Милан     | Зенит     |
    | Милан     | Спартак   |
    | Спартак   | Зенит     |
    | Спартак   | Милан     |
    

  * _home_team_ \- команда играет на своем поле
  * _away_team_ \- команда играет на гостевом поле



Команда не может играть сама с собой, поэтому значения в колонках не должны совпадать.
