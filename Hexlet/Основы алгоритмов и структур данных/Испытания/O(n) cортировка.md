# O(n) cортировка

## Задание

#  JavaScript: O(n) cортировка 

Основы алгоритмов и структур данных 6 сообщений

Начать испытание 

Упражнение можно решить на одном из языков программирования на выбор: JavaScript, PHP, Python или Java. Запишите решение в соответствующем файле в каталоге _solutions_.

**Проверка работает только для одного из языков, остальные файлы в директории _solutions_ должны остаться пустыми**

Реализуйте функцию, которая сортирует массив целых чисел по следующему правилу — сначала расставить все нечетные, а за ними все четные числа из массива. Постарайтесь написать решение не создающее новых массивов и сложностью O(n).

## solutions/solution.js

Функция принимает массив `arr` и возвращает отсортированный массив

Экспортируйте функцию по умолчанию.
    
    
    import solution from './solutions/solution.js';
    
    const arr = [3, 1, 12, 7, 8, 1, 6, 4];
    
    solution(arr)  // [3, 1, 7, 1, 8, 12, 6, 4]

    from solution import solution
    
    arr = arr = [3, 1, 12, 7, 8, 1, 6, 4]

## solutions/Solution.java

Создайте в файле пакет `solutions` с публичным классом `Solution`. Реализуйте в нем публичный статический метод `run()`, который принимает список `List` целых чисел и Метод должен вернуть список `List` целых чисел, отсортированных по указанному правилу
    
    
    int[] numbers = {3, 1, 12, 7, 8, 1, 6, 4};
    // Creating mutable list
    List<Integer> coll = new ArrayList<>(Arrays.asList(numbers));
    
    List<Integer> result = Solution.run(coll);
    System.out.println(result); // => [3, 1, 7, 1, 8, 12, 6, 4]
