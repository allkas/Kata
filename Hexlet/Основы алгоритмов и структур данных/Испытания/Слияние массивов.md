# Слияние массивов

## Задание

#  JavaScript: Слияние массивов 

Основы алгоритмов и структур данных 2 сообщения

Начать испытание 

Упражнение можно решить на одном из языков программирования на выбор: JavaScript, PHP, Python или Java. Запишите решение в соответствующем файле в каталоге _solutions_.

**Проверка работает только для одного из языков, остальные файлы в директории _solutions_ должны остаться пустыми**

Реализуйте и функцию, которая объединяет два отсортированных массива в отсортированный объединенный массив. Постарайтесь реализовать алгоритм с линейной сложностью `O(n+m)`, где n и m это длины массивов.

## solutions/solution.js

Функция принимает два параметра: отсортированные массивы `arr1` и `arr2`, и возвращает объединенный отсортированный массив

Экспортируйте функцию по умолчанию.
    
    
    import solution from './solutions/solution.js';
    
    const arr1 = [1, 3, 4, 5, 7];
    const arr2 = [2, 3, 5, 8, 12];
    
    solution(arr1, arr2);  // [1, 2, 3, 3, 4, 5, 5, 7, 8, 12]
    

## solutions/solution.php

Функция принимает два параметра: отсортированные массивы `arr1` и `arr2`, и возвращает объединенный отсортированный массив
    
    
    <?php
    
    $arr1 = [1, 3, 4, 5, 7];
    $arr2 = [2, 3, 5, 8, 12];
    
    solution(arr1, arr2);  // [1, 2, 3, 3, 4, 5, 5, 7, 8, 12]
    

## solutions/solution.py

Функция принимает два параметра: отсортированные списки `arr1` и `arr2`, и возвращает объединенный отсортированный список
    
    
    from solution import solution
    
    arr1 = [1, 3, 4, 5, 7]
    arr2 = [2, 3, 5, 8, 12]
    solution(arr1, arr2)  # [1, 2, 3, 3, 4, 5, 5, 7, 8, 12]
    

## solutions/Solution.java

Создайте в файле пакет `solutions` с публичным классом `Solution` и реализуйте в нем публичный статический метод `run()`. Метод принимает два параметра: отсортированные списки `List` целых чисел. Метод должен вернуть отсортированный объединенный список `List` целых чисел
    
    
    List<Integer> coll1 = List.of(1, 3, 4, 5, 7);
    List<Integer> coll2 = List.of(2, 3, 5, 8, 12);
    List<Integer> result = Solution.run(coll1, coll2);
    
    System.out.println(result); // => [1, 2, 3, 3, 4, 5, 5, 7, 8, 12]
