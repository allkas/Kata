# final, static, внутренние классы
## 1. final: где можно использовать
- **final class** — нельзя наследовать.
- **final method** — нельзя переопределить.
- **final variable** — нельзя изменить (примитив/ссылка).
**final vs static** — final константа, static принадлежит классу.
## 2. Может ли статический класс содержать нестатические поля?
Да, статический вложенный класс (static nested) — обычный класс, может иметь всё.
## 3. Типы вложенных классов
| Тип | Описание |
|-----|----------|
| Non-static inner | Имеет доступ к полям внешнего класса |
| Static nested | Не имеет доступа к нестатическим полям |
| Local class | Внутри метода |
| Anonymous class | Без имени, new Interface() {} |
