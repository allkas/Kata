# Сценарии видео — Карта

## Структура папки

Каждый файл = один готовый сценарий с полным разбором.

**Что есть в каждом сценарии:**
- Параметры (платформа, хронометраж, дата выхода)
- Психология — почему это видео сработает
- Превью / Обложка — что написать и показать
- Крючок — слово в слово + что в кадре
- Основной текст — по секундам, с монтажными пометками
- Субтитры — полный текст для копирайта
- Музыка — конкретные рекомендации
- CTA — несколько вариантов призыва
- Ошибки — чего не делать
