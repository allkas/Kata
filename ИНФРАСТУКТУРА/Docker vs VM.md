# 🏠 Docker vs виртуальные машины

## 🧠 Ментальная модель

| | **VM** | **Docker** |
|--|--------|------------|
| Образ | 🏠 Отдельный дом | 🏢 Квартира |
| Фундамент | Свой фундамент (kernel) | Общий фундамент (kernel хоста) |
| Размер | Гигабайты | Мегабайты |
| Запуск | Минуты | Секунды |

> [!important] Ключевое отличие
> **VM изолируют ОС целиком, Docker изолирует процессы на одном ядре.**

## 📊 Быстрая таблица

| Характеристика | VM | Docker |
|----------------|-----|--------|
| Гостевая ОС | Полноценная | Нет (ядро хоста) |
| Изоляция | Полная | Процессов |
| Кол-во на сервере | Десятки | Тысячи |
