# Брокеры сообщений
## Kafka
- Топики, партиции, офсеты.
- Pull-модель, хранение на диске.
- Высокая throughput.
## RabbitMQ/ActiveMQ
- Очереди, exchanges.
- Push-модель, сложная маршрутизация.
- Подтверждения (ACK).
## Разница
| Kafka | RabbitMQ |
|-------|----------|
| Потоки данных | Сообщения |
| Replay есть | Replay нет |
| Миллионы/сек | Тысячи/сек |
